# Todo List - Prueba Técnica
 
Esta es mi prueba técnica del bootcamp. He creado una todo list con temática de Camilo's Original Motor Works inspirada en GTA Online.
 
## Tecnologías usadas
 
- HTML
- CSS
## Lo que he hecho
 
He creado una interfaz de una aplicación de lista de tareas sin usar javascript. La aplicación tiene:
 
- Un header con el logo del taller
- Un formulario para añadir tareas nuevas
- Una lista con 5 tareas de ejemplo
- Botones para eliminar tareas completadas o todas las tareas
- Un footer con el nombre del taller
## Estructura del proyecto
 
```
todo-list/
├── index.html
└── css/
    └── styles.css
```
 
## Como se ve
 
La aplicación tiene una temática de taller de coches lowrider vintage con colores oscuros, dorado y rojo. He intentado que se parezca al estilo del taller Benny's de GTA Online pero con mi nombre.
 
## Responsive
 
He usado media queries para que se vea bien en:
 
- Móvil vertical
- Móvil horizontal
- Escritorio
## Cosas que he aprendido
 
- A usar HTML semántico con etiquetas como header, main, section, footer
- A hacer checkboxes personalizados solo con CSS
- A usar flexbox para maquetar
- A usar media queries para hacer la página responsive
- A usar nth-child para alternar colores en la franja del header
